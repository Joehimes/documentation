Introduction

Ruby in an interpreted, high-level, general-purpose programming language. It was designed and developed in the mid-1990's by Yukihiro "Matz" Matsumoto in Japan. It was developed to be thoroughly object-oriented, with inheritance, mixins, and metaclasses.

One of the big draws for many to the Ruby programming language is the Ruby on Rails web framework. This was designed to make setting up projects easier for the user to enable rapid application develpment.

What you should already know

This guide assumes you have the following basic background:

  • Basic computer skills.
  • Access to a text editor, i.e. Atom, VS Code, or Sublime (not a word proceesor).
  • Some terminal experience. Ruby programs are run from the command line.
