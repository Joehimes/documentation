Introduction

Ruby in an interpreted, high-level, general-purpose programming language. It was designed and developed in the mid-1990's by Yukihiro "Matz" Matsumoto in Japan. It was developed to be thoroughly object-oriented, with inheritance, mixins, and metaclasses.

One of the big draws for many to the Ruby programming language is the Ruby on Rails web framework. This was designed to make setting up projects easier for the user to enable rapid application develpment.

What you should already know

This guide assumes you have the following basic background:

  • Basic computer skills.
  • Access to a text editor, i.e. Atom, VS Code, or Sublime (not a word proceesor).
  • Some terminal experience. Ruby programs are run from the command line.

Hello world

Ruby comes pre-installed with a very handy program called Interactive Ruby (irb for short). This allows you to test snippets of code directly from the command line instead typing it into a ruby file and then running the program. It is also great for learning the basics of ruby programming. To begin an irb session, open the terminal and type irb. Once the session begins, type:

puts "Hello, World!"

Notice how it responds with "Hello, World!"? You have just written your first line of ruby! For now you can ignore the part where it also prints out nil. Every action in ruby will return something, even if there is nothing to return.

Variables

In ruby, there are five types of variables.

  • Global Variables
  • Instance Variables
  • Class Variables
  • Local Variables
  • Constants

Global Variables

Global variables are not generally recommended for use. They alter the global status of programs and can make them cryptic. They are defined using a $ followed by a name.

$global_variable = 10

Instance Variables

Instance variables begin with @. These are used within a class when initializing a new instance of that class.

          
  # Define Person Class

  class Person
    def initialize(name)
      @name = name
    end

    def say_hello
      puts "Hello! My name is #@name."
    end
  end

  # Create New Person
  person1 = Person.new("Joe")

  # Call Methods
  person1.say_hello   #"Hello! My name is Joe."

Class Variables

Class variables are declared with @@. These variables are shared among instances of that class.

Local Variables

Local variables are the most widely used. They begin with a lower case letter or _. The scope of these variables is limited to where they are defined and ranges from class, method, module, or within a block's braces.

Constants

Contants begin with an uppercase letter and are generally all uppercase letters to distinguish themselves. When defined within a class they can be accessed from within that class. If defined outside a class they can be accessed globally.
